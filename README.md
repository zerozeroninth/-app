
使用HTML5 Audio API创建的一个音乐可视化程序，



安装，需运行

```
git clone https://github.com/patrickroberts/music-js.git
cd music-js/static/
mkdir media
```

然后你需要放音乐文件到``media``文件夹，并把需要可视化的音乐文件名加到 ``index.html``中的JSON中。
之后再运行如下命令

```
cd ..
npm install
node app.js
```


然后打开 <http://localhost:8080>
